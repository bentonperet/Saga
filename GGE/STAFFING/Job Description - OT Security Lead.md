## Facility Introduction

Protect critical operations at Georgian Green Energy's Tbilisi facility, where operational technology security safeguards both a state-of-the-art data center and vital hydroelectric infrastructure. This unique integrated environment demands specialized expertise in both physical and cyber-physical security to defend against evolving threats to critical infrastructure.

## Job Summary

The OT Security Lead will manage all operational technology (OT) security and physical security operations for both the Data Center and Hydro Power Plant. This critical role protects industrial control systems, SCADA networks, and physical infrastructure from cyber and physical threats while ensuring business continuity and regulatory compliance.

## Reporting Structure

- **Reports to:** Chief Information Security Officer (CISO)
- **Direct Reports:** Security Guards/Officers, Access Control Operators, Physical Security Technicians
- **Key Relationships:** HPP engineers, Critical Infrastructure Engineer, data center operations, CITO, local law enforcement, emergency services

## Roles & Responsibilities

**OT/ICS Security:**
- Secure SCADA systems, PLCs, HMIs, and industrial control networks at hydro power plant
- Implement network segmentation and security zones per IEC 62443 standards
- Manage OT network monitoring and anomaly detection systems
- Coordinate secure remote access for vendors and maintenance personnel
- Oversee OT patch management and change control processes
- Conduct OT-specific vulnerability assessments and penetration testing

**Physical Security Operations:**
- Manage 24/7 physical security operations for both facilities
- Oversee security guard force and patrol operations
- Monitor CCTV surveillance systems and alarm monitoring
- Respond to and investigate security incidents and breaches
- Coordinate with law enforcement and emergency services
- Conduct security assessments and threat analysis

**Access Control:**
- Manage electronic access control systems (badge readers, biometrics)
- Administer employee, contractor, and visitor access privileges
- Oversee badging, credential issuance, and access termination processes
- Implement multi-layer security zones based on criticality
- Audit access logs and investigate unauthorized access attempts
- Coordinate data center customer access and escort procedures

**Perimeter Security:**
- Manage fence lines, gates, barriers, and vehicle access points
- Oversee intrusion detection systems and perimeter alarms
- Coordinate vehicle inspections and cargo screening
- Implement and test vehicle barriers and anti-ram measures
- Monitor and patrol dam and restricted areas

**SCADA & Control System Protection:**
- Implement secure architecture for power plant control systems
- Coordinate with HPP engineers on control system changes
- Manage firewalls and network security for OT networks
- Oversee secure remote monitoring and control capabilities
- Ensure separation between IT and OT networks
- Monitor for cyber threats targeting industrial control systems

**Security Systems Management:**
- Maintain CCTV cameras, recording systems, and video analytics
- Manage intrusion detection, motion sensors, and alarm systems
- Oversee door controllers, locks, and physical barriers
- Coordinate security systems integration and interoperability
- Ensure security systems uptime and redundancy
- Manage security equipment lifecycle and upgrades

**Compliance & Standards:**
- Ensure compliance with Georgian critical infrastructure protection regulations
- Implement NERC CIP standards (or European equivalents) for power systems
- Maintain alignment with IEC 62443 for industrial automation security
- Support data center security certifications (SOC 2, ISO 27001)
- Coordinate regulatory inspections and security audits
- Maintain security documentation and procedures

**Incident Response:**
- Lead security incident response team
- Investigate security breaches, intrusions, and suspicious activities
- Coordinate with CISO on cyber-physical incident response
- Document incidents and implement corrective actions
- Conduct post-incident reviews and lessons learned
- Coordinate with law enforcement on criminal investigations

**Training & Awareness:**
- Train security personnel on OT security and critical infrastructure protection
- Conduct security awareness training for plant operators and staff
- Run security drills and exercises (bomb threats, active shooter, cyber incidents)
- Develop security procedures and standard operating procedures
- Brief contractors and visitors on security requirements

## Required Qualifications

- Bachelor's degree in Security Management, Information Technology, Engineering, or related field
- Minimum 7 years experience in security operations, with at least 3 years in OT/ICS security
- Strong understanding of SCADA systems, PLCs, and industrial control systems
- Experience with physical security systems (access control, CCTV, intrusion detection)
- Knowledge of IEC 62443, NERC CIP, or similar critical infrastructure security standards
- Professional security certification (PSP, CPP, or GICSP)
- Excellent command of English and Georgian
- Experience in critical infrastructure (energy, utilities, or data centers)

## Desired Qualifications

- Advanced security certifications (CISSP, CRISC, GRID, GCIP)
- Background in law enforcement, military, or intelligence
- Experience with hydroelectric facilities or power generation security
- Knowledge of data center security standards and practices
- Familiarity with cybersecurity tools for OT environments
- Experience managing security guard forces
- Crisis management and emergency response training
- Project management experience for security system installations
- Russian language proficiency

## Experience Level

**Senior Professional** (7+ years experience, 3+ years OT/ICS security)
